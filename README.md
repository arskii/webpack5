# Webpack 5 React Boilerplate (React/Babel/SASS)

Webpack 5 React boilerplate using Babel, Sass with an production build and a hot development.

---

## Installation

Clone this repo and npm install.

```bash
npm i
```

## Usage

### Development Server

```bash
yarn start
# or
npm start
```

### Production Bundle

```bash
yarn build
# or
npm run build
```
